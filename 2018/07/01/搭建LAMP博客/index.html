搭建LAMP博客

发表于 2018-07-01 更新于 2021-04-11 分类于 软件程序

搭个人网站一般有两种选择：

  • GitHub Pages + Hexo / jekyl
  • 服务器 + WordPress / Typecho

我选择LAMP+阿里云+Wordpress。穷啊，阿里云学生有1折优惠，十块钱一个月。可以接受。 所谓LAMP，即：Linux+Apache+Mysql+Php 下面结合他人总结与自己实际遇到的问题，给大家介绍如何搭建自己的博客：

0x01 服务器选择 Ubuntu 16.04

0x02 购买ECS服务器后，可以同时做的事有：

  1. 购买域名，暂时还没买。原因：没钱！！！
  2. 下载 Xshell5（远程PowerShell），Xftp5（ftp传文件）。
  3. 看看喜欢什么WordPress插件。

0x03 使用 Xshell5 连接 ECS：

连接前先在把自己的服务器添加安全组，SHH协议默认连接22端口。添加具体方法大家可以百度谷歌。
